# Contributing to BenchKit

Thanks for your interest in contributing!

## How to contribute
- Fork the repository
- Create a new branch
- Make your changes
- Submit a pull request

## What you can contribute
- New hardware presets
- Improvements to scripts
- Bug fixes
- Documentation updates

## Code style
- Use clear, readable PowerShell
- Keep scripts safe and non-destructive
- Avoid registry edits unless necessary
